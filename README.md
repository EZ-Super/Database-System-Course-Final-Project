# 👥 Database System Course Final Project - G7

## 📒 Project Info
* 📌 Topic: 全方位電商平台智慧型資料庫系統
* 📋 Content:
  * 👤 用戶管理
    * 買家 & 賣家，會員等級、積分。 
  * 📜 商品管理
    * 分類、庫存、商品價格、供應商資訊、廣告推銷。
  * 🧾 訂單管理
    * 訂單狀態、支付、退貨退款。
  * 📦 物流與倉儲
    * 庫存分配、發貨時間預測。
  * 📊 數據分析
    * 用戶行為分析、熱銷商品、價格走勢。
